# Desktop Notifications for Claude Code

Get desktop notifications when Claude Code completes tasks. Walk away from your computer and get notified when Claude is done.

## Features

- Native desktop notifications on macOS, Windows, and Linux
- Triggers when Claude finishes responding

## Supported Platforms

- **macOS** - Native notification center
- **Windows** - Message box notification
- **Linux** - notify-send (requires libnotify)

## Installation

```
/plugin marketplace add ysaber/islamic-reminders-claude-plugin
/plugin install desktop-notifications@islamic-reminders-claude-plugin
```

Restart Claude Code after installation.

## Use Case

1. Ask Claude to run a long task (tests, builds, etc.)
2. Walk away to grab coffee
3. Get a desktop notification when it's done
