# Claude Plugins by Yusuf Saber

A collection of plugins for Claude Code.

## Installation

```bash
/plugin marketplace add yusufsaber/claude-plugins
```

## Available Plugins

### islamic-reminders

Display Islamic reminders and dhikr during Claude Code sessions.

```bash
/plugin install islamic-reminders@claude-plugins
```

**Features:**
- Shows Bismillah at session start
- Displays random dhikr after each response
